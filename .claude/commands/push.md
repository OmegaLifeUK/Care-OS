Push all changes to GitHub for the Care OS project.

Instructions:
1. Stage all changed and new files: `git add -A`
2. Check `git status` to see what's being committed
3. Create a commit with a concise message describing the changes
4. Push to `origin komal:main` (this is the OmegaLifeUK/Care-OS repo)
5. Log this action in `docs/logs.md`
6. Confirm to the user with the commit hash and what was pushed
